三步上手，命令行同样简单

1

安装

用 pip 一条命令装好工具箱

pip install mdtoolkit
2

选功能

目录 / 检查 / 排版 / 导出，按需调用

mdkit toc README.md
3

出结果

加 -w 写回文件，或 -o 指定导出路径

mdkit export post.md -o post.html

现在就把它装进你的电脑
